# Цифровой двойник преподавателя
Telegram-бот для автоматизации ответов на типовые вопросы студентов с использованием PostgreSQL и полнотекстового поиска.

## 📋 О проекте
Проект представляет собой интеллектуального Telegram-помощника, который:
- 📚 Ищет информацию в лекционных материалах
- 📖 Объясняет термины из глоссария
- ❓ Отвечает на частые вопросы (FAQ)
- 📝 Показывает решения типовых задач
- 📅 Сообщает расписание консультаций
- ⏰ Напоминает о дедлайнах
- 👨‍🏫 Перенаправляет сложные вопросы преподавателю

## 🎯 Целевая аудитория
- **Студенты** — быстрый доступ к учебным материалам и ответам на вопросы
- **Преподаватели** — снижение нагрузки на рутинные вопросы
- **Администрация вуза** — аналитика популярных запросов

## 🛠 Технологии
- **Python 3.11** — основной язык
- **python-telegram-bot 20.x** — библиотека для Telegram Bot API
- **PostgreSQL 15** — база данных с полнотекстовым поиском
- **asyncpg** — асинхронный драйвер для PostgreSQL
- **Docker** — контейнеризация
- **Git** — контроль версий
